Model and textures by Clifford Timm

Installation:

Unzip to your Trainset folder, a folder named "GATX-48ft tank" will be created
and the files added to it. Use the activity editor to add the car to the Sim.

Repaintable texture files will be available in a separate downloadable zip file.
