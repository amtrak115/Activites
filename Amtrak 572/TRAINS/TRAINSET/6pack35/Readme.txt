MSTS Set of six 50' boxcars for SP 243480, SP 243901, SP 246480, SP 247786, SSW 20430 and SSW 67546 in both loaded and empty versions.  Original model and textures by Ted Curphey. Truck and coupler texture by Jeff Auberpine. Repainted by Mark Giltz.  Photos provided by and used with the permission of Keith Belk, Thom McAdams, Rob Sarberenyi and Craig Walker.

email: markuprailfan@aol.com

JUNE 2011

Installation:

Unzip to your Trainset folder; a folder named "6PACK35" will be created and the files added to it.  Use the activity editor or ConBuilder to add the carS to the Sim.  The cars will be listed as "boxcar 50' SP 243480 Loaded", "boxcar 50' SP 243480 empty", etc.
